## Installation

#### downloading Project 

```
  git clone https://github.com/MmdBay/live-search-lyrics.git
```
#### Installation in production mode

```
  npm i
  cd Project/dist
  node app.js
```

## Run Locally

Clone the project

```
  git clone https://github.com/MmdBay/live-search-lyrics.git
```

Go to the project directory

```
  cd project
```

Install dependencies

```
  npm install
```

Start the server

```
  npx tsc -w
  cd dist
  nodemon app.js
```
